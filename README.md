# 418IATP-Pinus-Hack
A project from "418 I'm a teh poci" for Pinus Hack

# Instruction
1. Download this repository and extract anywhere in your computer.
2. Inside the newly extracted folder, open a terminal and create a virtual environment:
```
python -m venv venv
.\venv\Scripts\activate
```
3. Install the required modules:
```pip install -r requirements.txt```
